[title sub="Written by Chris Graham (ocProducts)"]Composr Tutorial: Composr keyboard shortcuts[/title]

Many functions in a Composr website may be activated by key code, rather than clicking or using tab to move between components. This short tutorial will list these key codes.

These key codes are based on the UK government standard for accessibility.

On most web browsers you can activate a key shortcut by holding down the Alt key and pressing the key code (hold down the Ctrl key on a Mac). You may need to hold down the Shift key in addition.
